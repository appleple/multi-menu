About MultiMenu.js

It enables you to create smooth sliding multi level navigation menu like native apps.

View it on GitHub
Installation

via CDN

<link rel="stylesheet" href="https://unpkg.com/multimenu@latest/css/multi-menu.css" />
<script src="https://unpkg.com/multimenu@latest/bundle/multi-menu.js"></script>

via npm

npm install multi-menu --save
Usage

js

document.addEventListener("DOMContentLoaded", () => {
  new MultiMenu('.js-multi-menu')
});
